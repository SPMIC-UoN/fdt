What's new

Low level modelling

  • Qboot - "Model-free" low level analysis using various Qball schemes
  • Improvements on bedpostx:
    • Option for noise floor modelling
    • Option for Rician noise modelling
    • Better nonlinear initialisation
    • Up to 20% improvement in processing speed
  • Nonlinear dtifit (with positivity constraints and Rician noise modelling)

Tractography

  • Combined surface/volume ROIs (for all types of ROIs in probtrackx)
  • Handle surfaces from Caret, FreeSurfer and FIRST
  • Storing various types of connectivity matrices
  • (Much) Better internal handling of memory

Misc tools

  • Surface projection
  • Surface to surface transformation
  • Matrix clustering tool for blind parcellation
